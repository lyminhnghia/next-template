## Getting Started

### First, install packages:

```bash
yarn
```

### Second, config env:

- Create new file .env
- Add this config to .env file:

```bash
NODE_ENV=development
API_DEV=
API_PROD=
```

### Last, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
